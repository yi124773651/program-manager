# V1.1.6 更新日志

## UI 多主题与切换

本次更新落地三套界面风格预设，并保持原有自定义主题色、背景图和窗口透明度语义不变。

## 主要变更

1. 新增三套界面风格：清新晨曦、深海黑曜、赤陶暖砂。
2. 自定义主题色独立保留，切换界面风格不会自动覆盖用户已选主色。
3. 旧配置缺少 `themePreset` 时会自动使用清新晨曦作为默认预设。
4. 背景图、背景遮罩和窗口透明度可继续与主题预设叠加使用。
5. 主窗口、快捷搜索、待办日程表和快捷便签统一使用同一套主题服务。

## Workflow 适配

1. 发布版本已统一到 `1.1.6`。
2. `v1.1.6` tag 可通过 release workflow 的版本匹配检查。
3. 前端、Rust 测试和格式检查继续作为发布门禁。

## 验证重点

1. 设置页可切换界面风格，并可单独应用该风格推荐主色。
2. 快捷搜索、待办日程表、快捷便签窗口打开后使用当前主题。
3. 导入本地数据触发配置刷新后，主题状态跟随最新配置更新。
